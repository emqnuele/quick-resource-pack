# Quick Resource Pack 1.1.0 - Modrinth Release Text

Use one Version entry per jar on Modrinth.
Each section below already has a title and changelog for that exact Minecraft patch.

## Shared Release Description (optional long text)

Quick Resource Pack 1.1.0 is a major step forward: the mod now includes its own built-in menu, instead of relying entirely on Mod Menu for quick pack management.

Highlights:
- Added a dedicated in-mod menu to select and manage the resource pack used for quick toggling.
- Improved integration in the vanilla Resource Packs screen.
- Stabilized compatibility across the full Minecraft 1.21.x line with patch-specific builds.
- Improved overall reliability on newer 1.21.x patches.

Included fixes:
- Fixed invisible pack names in the list on 1.21.8+.
- Fixed Set and Cancel buttons not responding in the Select Pack menu on newer patches.
- Fixed startup crash on 1.21.11 caused by keybind initialization.
- Improved resilience against minor API differences between 1.21.x patches.

Important notes:
- Builds are patch-specific: download the jar matching your exact Minecraft version.
- Required dependencies: Fabric API, Cloth Config, Mod Menu.
- Required Java version: 21.
