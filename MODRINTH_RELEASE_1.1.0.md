# Quick Resource Pack

## Stable Release 1.1.0

Toggle one selected resource pack on or off instantly with a keybind, without restarting the game.
This release adds a built-in quick menu and improves integration with the vanilla Resource Packs screen.

---

## Full Description

Quick Resource Pack is a client-side Fabric mod that lets you pick a resource pack once and toggle it anytime while playing.

Version 1.1.0 is a major update:
- New built-in mod menu for quick pack selection
- Better vanilla Resource Packs integration
- Improved compatibility and stability across 1.21.x patch versions

## Features

### Quick Toggle
- Toggle your selected pack with a configurable keybind (default: R)
- Asynchronous loading to reduce in-game freezes
- Toast notifications for loading, enabled, and disabled states

### Built-in UI
- Dedicated in-mod quick menu
- Set for Quick Toggle button in the vanilla Resource Packs screen
- Fast pack picker workflow for daily use

### Configuration
- Select target resource pack
- Enable or disable notifications
- Optional auto-apply on game start
- Cloth Config + Mod Menu integration

## Requirements

- Minecraft 1.21.x (patch-specific files are provided)
- Java 21
- Fabric Loader
- Fabric API
- Cloth Config API
- Mod Menu

## Usage

1. Open the mod quick menu (or config screen).
2. Select the resource pack you want to bind.
3. Use the keybind in-game to toggle it on or off.

Alternative flow:
Open vanilla Resource Packs, select a pack, then click Set for Quick Toggle.

## Compatibility

- Client-side only (no server installation required)
- Works well with common client-side mods (including Sodium and Iris)
- Patch-specific 1.21.x builds are provided for better stability per Minecraft version

## Shared Release Description (optional long text)

Quick Resource Pack 1.1.0 is a major step forward: the mod now includes its own built-in menu, instead of relying entirely on Mod Menu for quick pack management.

Highlights:
- Added a dedicated in-mod menu to select and manage the resource pack used for quick toggling.
- Improved integration in the vanilla Resource Packs screen.
- Stabilized compatibility across the full Minecraft 1.21.x line with patch-specific builds.
- Improved overall reliability on newer 1.21.x patches.

Included fixes:
- Fixed invisible pack names in the list on 1.21.8+.
- Fixed Set and Cancel buttons not responding in the Select Pack menu on newer patches.
- Fixed startup crash on 1.21.11 caused by keybind initialization.
- Improved resilience against minor API differences between 1.21.x patches.

Important notes:
- Builds are patch-specific: download the jar matching your exact Minecraft version.
- Required dependencies: Fabric API, Cloth Config, Mod Menu.
- Required Java version: 21.
